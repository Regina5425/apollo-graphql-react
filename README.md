## About The Project

A simple todo app created by Apollo & GraphQL

### Built With

- ReactJS
- Apollo Client
- GraphQL
- json-graphql-server

<!-- GETTING STARTED -->
### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Regina5425/apollo-graphql-react
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Run the app on your localhost
   ```js
   npm run start
   ```
